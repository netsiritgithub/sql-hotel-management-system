# 🏨 Hotel Management System (SQL + Oracle APEX)

## 📌 Overview
This project is a relational database system designed to manage hotel operations including guest bookings, room assignments, staff management, services, maintenance, payments, and feedback.

## 🧰 Technologies Used
- Oracle SQL
- Oracle APEX
- Relational Database Design

## 🗂️ Database Structure
The system includes 10 interconnected tables:
- Department
- Staff
- Room
- Guest
- Booking
- Feedback
- Hotel_Service
- Service_Request
- Payment
- Maintenance

## ⚙️ Key Features
- Manage guest bookings and stay history
- Track room assignments and availability
- Handle service requests and maintenance tasks
- Record payments and financial transactions
- Store guest feedback and ratings

## 📊 Sample Queries
- Retrieve guest booking history
- Track payments by guest
- Analyze service requests
- Calculate total revenue
- Evaluate customer feedback ratings

## 🖥️ Oracle APEX Application
An interactive web application was built using Oracle APEX featuring:
- Interactive reports for data visualization
- Forms for inserting and updating records
- User-friendly interface for hotel operations
